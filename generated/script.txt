Introduction:
Singular Value Decomposition (SVD) is a powerful matrix factorization technique that decomposes any matrix *A* (m x n) into three matrices: *U*, *Σ*, and *V<sup>T</sup>*.  *U* is an *m x m* orthogonal matrix whose columns are the left singular vectors, *Σ* is an *m x n* diagonal matrix containing the singular values of *A*, and *V<sup>T</sup>* is an *n x n* orthogonal matrix whose columns are the right singular vectors. This decomposition reveals the rank, principal components, and the amount of variance within the data represented by the matrix. It's a fundamental concept in dimensionality reduction and various applications like Latent Semantic Analysis.

Core Explanation:
Singular Value Decomposition (SVD) is a mathematical technique that decomposes a matrix *A* (m x n) into three matrices: *U*, *Σ*, and *V<sup>T</sup>*.

*   **U** is an *m x m* orthogonal matrix whose columns are the left singular vectors of *A*. These vectors span the column space of *A*.
*   **Σ** is an *m x n* diagonal matrix containing the singular values of *A*, which are non-negative and arranged in descending order. These singular values represent the "strength" of the corresponding singular vectors.
*   **V** is an *n x n* orthogonal matrix whose columns are the right singular vectors of *A*. These vectors span the row space of *A*.

The singular values are the square roots of the eigenvalues of *A<sup>T</sup>A* and *AA<sup>T</sup>*. The rank of the matrix *A* is equal to the number of non-zero singular values. SVD is used to determine the rank, principal components, and dimensionality of the data represented by the matrix, and is fundamental for dimensionality reduction techniques.
